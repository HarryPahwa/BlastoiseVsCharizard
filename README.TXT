Read this file before playing the game

Let the game load up. You should hear music when it loads up.

Please watch the video first.

Controls are WASD to control the Pokemon, space to jump,
q to attack, and e to defend.

Use mouse to aim. Do not mouseclick.

Please make sure the window that displays the health information and 
other details is in focus.
See the screenshot attached.
Please make sure only one monitor is connected, else there might be an error.

Capslock being off is recommended.
